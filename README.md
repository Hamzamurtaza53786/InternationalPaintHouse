🎨 International Paint House (IPH) – E-Commerce App

A modern Flutter-based e-commerce application for paint products, color selection, and home improvement services.
Built for International Paint House (Karachi) with 50+ years of business legacy.

🚀 Features
🛒 E-Commerce System
Product listing with categories
Brand-wise filtering
Product detail pages (SEO friendly URLs)
Cart system with price calculation
🎨 Paint Tools
Paint estimation calculator (area-based)
Product coverage suggestion system
Shade & color preview system
💬 Customer Support
WhatsApp integration
Email support
Contact page with full business info
📱 Responsive UI
Mobile + desktop adaptive design
Clean Material UI layout
Fast performance optimized UI
